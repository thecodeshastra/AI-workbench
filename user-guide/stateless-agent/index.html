Stateless Agent Guide

The Stateless Agent is the simplest agent in the workbench. It is designed to handle direct, one-off user requests without maintaining conversational history or complex state.

How it Works

The agent takes a user's input, wraps it in a predefined prompt template (AGENT_PROMPT), and sends it to the configured LLM provider. The response is then returned directly to the user.

Key Features

  • Fast & Lightweight: No overhead for state management or complex planning.
  • Direct: ideal for "Ask and Answer" scenarios.
  • Code Generation: Optimized for generating python code snippets or answering technical questions.

Usage Example

To use the stateless agent, run the provided CLI script:

python3 src/run_stateless_agent.py

The agent will start an interactive session where you can enter your requests:

Stateless AI Agent (Multi-Provider)
----------------------------------

You: Write a Python function to calculate the Fibonacci sequence.

Agent:

[Agent generates response here]

You: exit

Type exit or quit to terminate the session.
